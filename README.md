# Netflix Data Analysis

Exploratory Data Analysis (EDA) of the Netflix Movies and TV Shows dataset.

## What I explored
- Distribution of Movies vs TV Shows
- Top countries producing Netflix content
- How Netflix's content mix changed over the years (2017-2021)
- Most common genres/categories on Netflix

## Tools used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key findings
- 70% of Netflix catalog is Movies, 30% TV Shows!
- United States dominates content, India is Second!
- Netflix shifted toward TV Shows by 2021
- Dramas + International Movies is the most common genre combination overall.
